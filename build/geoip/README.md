# GeoIP database

`dbip-country-lite.mmdb` is the **DB-IP IP-to-Country Lite** database, used for
peer country flags in the Peers tab. Lookups are fully offline — no peer IP ever
leaves your machine.

IP Geolocation by DB-IP (https://db-ip.com) — licensed under
[CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

Refresh monthly with `scripts/update-geoip.sh` and commit the updated file.
